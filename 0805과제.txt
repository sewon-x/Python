숙제 개요 :  Boston Housing Price 데이터를 읽고, 간단한 통계 분석하는 Python 프로그램을 작성해 봅시다. (제출마감: 8월 6일 00시)

 

숙제 내용: 다음의 동작을 수행할 수 있도록 Python 코드를 작성합니다.

1.     Boston Housing Price 데이터 (https://ai-times.tistory.com/431) 에서 다운로드 하여, 엑셀에서 파일-> 다른 이름으로 저장-> .csv 파일로 저장을 선택하여 데이터를 CSV로 구성하고, 파이썬 코드에서 파일을 읽을 수 있도록 합니다(5점)

2.     해당 csv 파일을 읽어, MEDV 필드의 값에 대한 평균과 최대, 최소 값을 계산하여 각각 Prompt 상에 출력합니다. (10점)
(출력 형식: avg. = 숫자, max = 숫자, min = 숫자)

3.     해당 csv 파일을 읽어, MEDV 필드의 값이 최대인 행과 최소인 행을 찾아내고, 이들의 CRIM, DIS, TAX 값을 비교 출력합니다.(10점)
(출력 형식: 가장 비싼 집들은 각각 CRIM =숫자, DIS=숫자, TAX =숫자 이고, 가장 싼집은 각각 CRIM=숫자, DIS=숫자, TAX=숫자 입니다)

4.     2. 3의 결과들을 text 파일로서 output.txt로 출력하도록 합니다. (5점)
